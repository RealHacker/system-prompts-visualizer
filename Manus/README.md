# Manus Agent System Prompt Visualization

This visualization provides an interactive exploration of the Manus AI agent's system prompt, showcasing its capabilities, workflow, tools, and operational rules.

## Files in this directory

- `Prompt.txt` - Main system prompt document
- `Modules.txt` - Detailed module specifications with XML-like structure
- `Agent loop.txt` - Description of the agent's operational loop
- `tools.json` - JSON definition of all 29 available tools
- `system_prompt_visualizer.html` - Main visualization interface
- `styles.css` - Styling for the visualization
- `script.js` - Interactive functionality
- `analyze_prompt.py` - Python script used to analyze the prompts
- `prompt_analysis.json` - Analysis results from the Python script

## Key Features of the Visualization

### Overview
- Total word count: 3,826 words
- Number of tools: 29
- Format: Mixed (Plain text, XML-like tags, JSON)

### Sections Covered
1. **Overview** - Agent role and capabilities
2. **Agentic Workflow** - Step-by-step operational process
3. **Tools** - Detailed descriptions of all 29 tools
4. **Context Management** - How the agent handles information flow
5. **Rules** - Operational guidelines and constraints
6. **Coding Standards** - Development environment and practices
7. **Other Information** - Limitations and system modules

### Visualization Elements
- Interactive sidebar navigation
- Statistics dashboard with key metrics
- Word distribution visualization
- Responsive design for all devices

## How to Use
Simply open `system_prompt_visualizer.html` in any modern web browser to explore the Manus Agent system prompt interactively.
